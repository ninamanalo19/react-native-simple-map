# react-native-simple-map
Simple Map React-Native Component

A work in progress...

# Properties

Property  | Type | Valid Values | Optional or Required
---------  | -----| ------------ | --------------------
showUserLocation | boolean | true or false | Optional
markers | Array | See sample below | Optional
